# CheerRing

CheerRing is a pro wrestling fan support landing page powered by CheerLink.

Public page:
https://manabu619.github.io/cheerring/

## Files

- `index.html`: landing page
- `hero_bg2.webp`: optimized hero background used by the page
- `hero_bg2.png`: OGP image fallback
- `404.html`, `manifest.json`, `robots.txt`, `sitemap.xml`: static site support files

## Current Note

The early access form is ready for Formspree, but the real Formspree ID is not configured yet. Until then, submissions fall back to email.
